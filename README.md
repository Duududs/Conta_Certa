#### 「 Visão geral 」

O **Conta Certa** é uma aplicação front-end desenvolvida para ajudar no gerenciamento de contas mensais de forma simples, visual e eficiente.  
O projeto permite acompanhar pagamentos, vencimentos e valores pendentes por mês, oferecendo uma visão clara da situação financeira.

O foco principal está em **UX/UI de produto**, organização de estado e responsividade, simulando um dashboard real utilizado no dia a dia.

---

#### 「 Problema que resolve 」

Controlar contas mensalmente costuma ser confuso quando feito por anotações ou planilhas pouco intuitivas.

O **Conta Certa** resolve esse problema ao:

- Centralizar todas as contas do mês em um único lugar
- Destacar rapidamente valores pagos, pendentes e atrasados
- Oferecer um resumo visual claro para tomada de decisão

---

#### 「 Funcionalidades 」

- Navegação por mês (anterior / próximo)
- Cadastro, edição e remoção de contas (CRUD completo)
- Marcar contas como pagas ou pendentes
- Identificação automática de contas atrasadas
- Filtros:
  - Todas
  - Pagas
  - Pendentes
  - Atrasadas
  - Vencendo em até 7 dias
- Resumo financeiro mensal:
  - Total previsto
  - Total pago
  - Total pendente
- Alternância entre **modo claro e modo escuro**
- Persistência de dados com **localStorage**
- Layout totalmente **responsivo** (mobile, tablet e desktop)

---

#### 「 UI / UX 」

- Design premium em dark mode inspirado em dashboards modernos
- Versão light mode limpa e elegante
- Hierarquia visual clara
- Componentes reutilizáveis
- Estados vazios pensados para guiar o usuário
- Microinterações e feedback visual

---

#### 「 Como rodar o projeto localmente 」

```bash
git clone https://github.com/Duududs/Conta_Certa.git
cd conta-certa
npm install
npm run dev
```
